WebRTC Instructions:

  1. Open this page in two different browsers (Chrome + Firefox, or Chrome + Edge)
  2. Click "WebRTC Mode" in both
  3. Copy the Room ID from one tab and paste it in the other
  4. Click "Connect" to establish P2P connection

Server Mode Instructions:

  1. Run: git clone https://github.com/arcodange-org/sync-audio-js.git
  2. Then: cd sync-audio-js && npm install && npm start
  3. Open: http://localhost:8080/example/ in multiple tabs
