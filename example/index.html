🌐 Server Mode

Sync across network using WebSocket (requires server)

📋 Server Mode Setup

  1. Clone: git clone https://github.com/arcodange-org/sync-audio-js.git
  2. Install: cd sync-audio-js && npm install
  3. Start: npm start
  4. Open: http://localhost:8080/example/ in multiple tabs
